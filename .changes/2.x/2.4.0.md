
## 2.4.0 - 2026-05-11

### Added

- support to PHP 8.5.6
- support to PHP 8.4.21
- support to PHP 8.3.31
- support to PHP 8.2.31
- support to `Mago` PHP toolchain (https://github.com/carthage-software/mago)
- support to `bat` cli tool (https://github.com/sharkdp/bat)
- support to `hyperfine` cli benchmarking tool (https://github.com/sharkdp/hyperfine)
- support to PHPUnit 13

### Changed

- upgrade [docker-php-extension-installer](https://github.com/mlocati/docker-php-extension-installer) to version 2.11.1
- raise Symfony components minimum requirement constraint (to new LTS 7.4)
- add compatibility for Symfony components v8
- disable `swoole` on PHP 8.2, 8.3, 8.4, 8.5

**Full Changelog**: [2.3.0...2.4.0](https://github.com/llaville/docker-php-toolbox/compare/2.3.0...2.4.0)
